EasyDateTime examples v1.0.0
All input dates are expected in the UTC format, the methods will try to convert them into the timezone of the object. If you don't want this behaviour you need to set the timezone property to 'UTC'.
$edt = new EasyDateTime('Asia/Tehran', 'jalali');
$edt->date();
Gets the current datetime - no need for any parameters!
